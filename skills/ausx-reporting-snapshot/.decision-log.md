# Decision Log: ausx-reporting-snapshot

## 2026-05-20

- Classified as a module-based simple workflow because it creates bounded report artifacts and should not maintain a conversational persona.
- Made the source contract strictly read-only to keep stakeholder reporting from mutating the knowledge base it summarizes.
- Added both Markdown and JSON outputs so MVP1-style UI ingestion can start without changing the human workflow.
- Required explicit blocking/advisory separation because visibility artifacts lose value if they flatten governance risk.
