# BMAD Integration Notes

LENS is a BMAD module and uses BMAD-native `bmad-lens-*` skill names.

## BMAD Owns

Analysis workflows, planning workflows, product brief, PRFAQ, PRD, UX, architecture, epics, stories, sprint planning, story creation, story implementation, code review, correct-course, and retrospective.

## LENS Owns

Discovery, capture, extraction, context sufficiency, relationship modeling, slice selection, capability derivation, impact analysis, workstream awareness, traceability, landscape evolution, focused BMAD packet preparation, BMAD artifact sync, story guarding, validation, Salmon correction, Doctor audits, and Auspex visibility.

## Handoff Pattern

1. LENS discovers and frames the active slice.
2. LENS prepares a focused BMAD packet.
3. BMAD performs formal planning and implementation.
4. LENS syncs BMAD artifacts back into traceability.
5. LENS guards stories and validates slice, journey, and outcome evidence.
6. LENS raises Salmon when implementation reveals upstream changes.
